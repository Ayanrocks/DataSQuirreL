# DataSQuirreL

An lightweight, full fledged database viewer with a modern and sleek UI

# Running

To run the project, use the following commands

```sh
# Install Cargo tauri CLI
cargo install tauri-cli
npm i
cargo tauri dev
```

## Fetaures In v1.0.X

- View Tables
- Write Queries
- CRUD on Tables
- Support for multiple databases
  - PostgreSQL
  - MySQL
  - MSSQL

## Features in v1.1.X

- Add support for logging queries
- Connect with multiple databases

## Features in v2.0.X

- Theming support with five themes
  - Material UI
  - Ayu
  - Dracula
  - Noctis
  - Solarized
