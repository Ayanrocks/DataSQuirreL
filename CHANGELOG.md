## [0.1.3] - 2026-03-02

### Added

- Core database viewer functionalities.
- Support for PostgreSQL, MySQL, and MSSQL.
- CRUD operations directly from the user interface.
- Advanced querying with fuzzy search mechanisms.
- Undo/Redo historical operations for cell edits in the datatable.
- Keyboard navigation (Excel-like UI) for robust usage.
